GENERAL DESCRIPTION
	This program solves numerical problems of Electric Charges and Fields Unit 1 of 
Physics  textbook of class XII CBSE. This can be extended to solve numericals of other
subjects.

RUN FLOW
	The program first prompts a user manual, then after pressing any key it proceeds
to main menu. The menu has four options: 
1. Summary of the unit
2. Problem Solver
3. Help on how TUT can help you
4. Exit
